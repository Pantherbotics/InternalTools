Our Github organization for offline code editing can be found at https://github.com/Pantherbotics/

The PVCS (Pantherbotics Version Control System) is a web-based graphical frontend for the all-powerful git, a very well-known piece of version control software. Use this webpage to create a new repository and view existing ones. If a repository is created on Github first, and you want it on the local server, create a new local repo with the same name (Capitalization Counts!) as the one on Gihub, and then manually re-synchronize the repos. The commits should copy over.

WARNING: Blank Repositories that have been created but never committed to will be permanently deleted after three days!!! Ask Roger, Josh, or anyone else with server access if you need a repository deleted manually.
